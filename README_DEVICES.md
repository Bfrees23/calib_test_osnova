# Система управления приборами

Веб-приложение для работы с измерительными приборами (MIT, калибраторы M90, корректоры LKG).

## Стек технологий

- **PHP 8.3** - серверная часть
- **NGINX** - веб-сервер
- **Docker** - контейнеризация
- **JavaScript (Vanilla)** - клиентская часть
- **CSS3** - стилизация

## Структура проекта

```
src/
```

## Функционал

### MIT (Многоканальный измеритель температуры)
- Подключение/отключение к устройству
- Чтение температурных данных с 3 каналов
- Отображение температуры в Цельсиях и Кельвинах
- Проверка даты поверки устройства

### Калибраторы M90
- Подключение до 3 калибраторов одновременно
- Проверка подключения и получение серийного номера
- Чтение текущей температуры
- Управление калибровкой:
  - Установка температуры калибровки
  - Запуск/остановка процесса калибровки
  - Контроль состояния регулятора
  - Отслеживание времени стабилизации

### Корректоры LKG
- Подключение до 3 корректоров одновременно
- Настройка скорости соединения (9600-115200 бод)
- Чтение паспорта устройства
- Инициализация LKG ключа
- Получение данных из регистров

## Запуск проекта

1. Убедитесь, что Docker и Docker Compose установлены

2. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd <project-directory>
```

3. Запустите проект:
```bash
docker compose -f dev.yml --profile client up -d --build
```

4. Откройте в браузере:
```
http://localhost
```

## API

Все запросы к API осуществляются через `/api/devices.php` с параметром `action`.

### Доступные действия:

#### Общие
- `getPorts` - получение списка доступных COM-портов
- `getStatus` - получение статуса всех устройств

#### MIT
- `connectMIT` - подключение к MIT
- `disconnectMIT` - отключение от MIT
- `readMIT` - чтение данных температур

#### Калибраторы
- `connectCalibrator` - подключение к калибратору
- `disconnectCalibrator` - отключение калибратора
- `checkCalibratorConnection` - проверка подключения
- `obtainCalibratorData` - получение данных
- `startCalibration` - запуск калибровки
- `stopCalibration` - остановка калибровки

#### Корректоры
- `connectCorrector` - подключение к корректору
- `disconnectCorrector` - отключение корректора
- `checkCorrectorConnection` - проверка подключения
- `obtainCorrectorData` - получение данных

## Протоколы обмена

### Калибратор M90
- Команды отправляются с CRC16
- Основные команды: `:`, `!`, `T`, `!R76`, `!R0`, `!R63`, `!W76`, `!W63`
- Ответы содержат данные в HEX формате

### Корректор LKG
- Протокол Modbus RTU
- Функции: 0x03 (чтение), 0x04 (чтение входных регистров), 0x10 (запись), 0x11 (ответ)
- CRC16 Modbus

### MIT
- Данные передаются в формате: `1:25.5 B 2:26.3 B 3:24.8 B`
- NULL данные: -1000000.0

## Требования к оборудованию

- COM-порты для подключения устройств
- Для работы с реальными устройствами требуется PHP расширение для работы с последовательным портом
